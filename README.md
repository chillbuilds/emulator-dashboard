# Xemu Dashboard

Built with:
- nodejs
- express
- fs
- socketio

Requirements:
- xemu xbox emulator
- <a href="https://vimm.net/vault/Xbox">xbox roms</a> 

adding a new game:

- add game to xemu game folder
- box art must be .avif (found on <a href="https://vimm.net/vault/Xbox">Vimms Lair</a>)
- add box art to box art folder
- restart the server
